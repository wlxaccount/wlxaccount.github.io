3本地博客发布到Github Pages

第一步，需要在Github上创建好Github Pages仓库。Github注册以及Github Pages创建
第二步，安装发布插件。
在站点目录中运行命令行输入npm install hexo-deployer-git --save
后续博客的修改等等各种功能 都需要安装各种插件。这个对博客的帮助是非常大的。
等待插件安装完成

3.1将本地目录与Github关联起来。

命令行输入ssh-keygen -t rsa -C "邮箱地址"，这里要输入之前注册Github时的邮箱，例如我之前注册用的是 example@163.com，那命令行就输入ssh-keygen -t rsa -C "example@163.com"输入后一直回车。按提示的路径找打 id_rsa.pub 文件并复制其内容到 Github 点击右上角的头像 Settings 选择SSH and GPG keys 点击New SSH key 将之前复制的内容粘帖到Key的框中。 上面的title 可以随意 点击add 完成添加。
